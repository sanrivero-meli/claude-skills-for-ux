# Rapid Prototype

## What it does

Scaffolds production-ready React prototypes with pre-decided stack choices:
- shadcn/ui with dark mode
- Inter font
- framer-motion animations

## How to use

1. Install the skill
2. Describe the component or page you need
3. Claude scaffolds the structure, then fills in your feature logic

## Stack decisions (pre-made)

- **UI**: shadcn/ui
- **Theme**: Dark mode by default
- **Font**: Inter
- **Animation**: framer-motion
