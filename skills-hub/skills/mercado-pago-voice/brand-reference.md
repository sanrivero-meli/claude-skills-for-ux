## 1. Yellow Character (Core Identity Engine)

### Propósito
Conectar personas para que ganen juntas. El éxito compartido es mejor que el éxito individual.

### Mantra
**"Tu éxito es mi éxito."**

### Historia
Me gusta ganar. Pero entendí que es mejor cuando ganamos todos. Pasamos de una lógica gana–pierde a una lógica gana–gana.

### Creencia
Siempre buscamos dinámicas win-win. El crecimiento debe ser compartido.

### Motivación
Unir personas con un propósito común. Cuando perseguimos el mismo objetivo, aparecen mejores resultados.

## 2. Voice Principles (Sistema Base)

### Cómo deben ser los mensajes

| Principio | Descripción | Regla operativa |
|-----------|-------------|-----------------|
| Positivo | Siempre construimos desde el progreso | Nunca comunicar desde miedo o pérdida |
| Energético | Ritmo alto, lectura rápida | Titulares cortos |
| Sintético | Decir mucho escribiendo poco | Evitar párrafos largos |
| Accionable | Siempre invitar a hacer algo | CTA claro |

- Ser exitoso
- Desafiar el status quo
- Tener fe en el futuro

## 6. CX Emotional Thermometer (Sistema de Respuesta)

### Irritación
- Frases breves
- Reconocer frustración
- Solución concreta

**Ejemplo**: "Entiendo que esto puede ser frustrante. Vamos a resolverlo paso a paso."

### Miedo
- Tono calmado
- Explicación clara
- Seguridad primero

**Ejemplo**: "Quedate tranquilo, estamos revisando tu caso y te vamos a acompañar."

### Alegría
- Mantener energía
- Conversación ágil
- Refuerzo positivo

**Ejemplo**: "¡Qué bueno verte por acá! Lo resolvemos enseguida."

## 7. Sistema de Reglas Operativas

### Validaciones automáticas
- Máximo 1 emoji por mensaje
- CTA obligatorio en marketing
- Evitar frases largas (>25 palabras)
- Prohibido tono alarmista
- Siempre enfoque win-win

### Tono recomendado por contexto

```
if contexto == "error":      usar = "Empático y Justo"
if contexto == "promo":      usar = "Enérgico + Motivador"
if contexto == "educativo":  usar = "Adaptable"
if contexto == "deuda":      usar = "Empático con responsabilidad"
```
