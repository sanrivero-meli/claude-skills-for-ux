# Mercado Pago Voice

## What it does

Ensures all written content aligns with Mercado Pago's brand voice and tone guidelines. Works for:

- **UX copy** — buttons, modals, error messages, onboarding flows
- **Marketing text** — campaigns, landing pages, emails
- **CX responses** — support replies, help center articles

## How to use

1. Install the skill
2. Ask Claude to write or review copy using the Mercado Pago voice
3. Get brand-compliant content with tone suggestions and alternatives

## Capabilities

- Write new copy from a brief or description
- Review existing copy for brand voice compliance
- Suggest tone adjustments to match MP guidelines
- Flag off-brand language and provide alternatives
