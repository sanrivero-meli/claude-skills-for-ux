# Brief Generator

## What it does

Transforms raw, unstructured input — especially voice notes and verbal transcripts — into a complete, session-ready design brief following the team's standard template.

- Extracts signal from non-linear, repetitive transcripts
- Resolves contradictions and surfaces implicit assumptions
- Outputs a structured brief with: contexto, objetivos, value prop, KPIs, tono, and narrative

## How to use

1. Install the skill
2. Paste a voice note transcript, meeting notes, Slack thread, or verbal feature description
3. Get a complete brief ready for team alignment sessions

## Output includes

- **Contexto / Problema** — pain points and target segment
- **Objetivos y value props** — business and experience goals with KPIs
- **Scope** — what's being designed, milestones, and impacts
- **Moods y tono** — emotional arc and communication angles
- **Historia a contar** — narrative north star for the design team
- **Para revisar en sesión** — open questions for team alignment
